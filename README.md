# Portfolio — Anthony Vallone

My front-end developer portfolio site, built from scratch in a single HTML file.

## Features
- Scroll-reveal animations using IntersectionObserver
- Fixed navigation with backdrop blur effect
- Responsive layout — mobile-first design
- Skills grid with animated progress bars
- Project cards linking to live demos and GitHub repos
- Contact section with email, LinkedIn, and GitHub

## Tech Stack
HTML5 · CSS3 · JavaScript · Google Fonts · CSS Custom Properties

## Design
Dark editorial aesthetic — custom CSS design system with no frameworks or templates.
Inspired by high-end print design applied to web.
